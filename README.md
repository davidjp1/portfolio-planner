# portfolio-tracker

## Prerequisites
- node.js v14
- An alpha vantage API key [https://www.alphavantage.co](https://www.alphavantage.co)

## Setup
- Create a new file in the base directory called `.env.local` and add `REACT_APP_ALPHA_VANTAGE_KEY=<YOUR_KEY_HERE>` to it
- Run `npm install`
- Run `npm start`
